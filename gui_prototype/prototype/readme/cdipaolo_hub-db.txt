b'## `hub-db`\n#### A PornHub images meta-data dataset\n\n`hub-db` is a dataset of information about albums in the adult website [PornHub](pornhub.com). This application crawls through the \'most viewed\' search pages (which pages are defined in the [`config.json`](config.json) file) and recursively crawls all albums on those pages, and the images from those albums. No images are saved, but links to the images as well as tag metadata, upload timestamp, comments, etc. are saved.\n\nThis repository includes both the code to crawl PornHub to get this data as well as the dataset itself (when crawling is done).\n\n### JSON Data Structure\n\nBelow is the structure for the json, which is dumped by BasePage into separate files. These files can be then combined to form whatever smaller and more specific dataset you want.\n\nExample JSON dumps can be found in [`data`](data/), along with the actual dataset fully compiled.\n\n```go\n// BasePage corresponds to the most viewed\n// photos page at a given page number\ntype BasePage struct {\n\tURI    string  `json:"uri"`\n\tPage   uint64  `json:"page"`\n\tAlbums []Album `json:"albums"`\n}\n\n// Album corresponds to a image album\ntype Album struct {\n\tTitle          string  `json:"title"`\n\tURI            string  `json:"URI"`\n\tVotes          uint64  `json:"votes"`\n\tUpvotePercent  float64 `json:"upvote_percent,omitempty"`\n\tViews          uint64  `json:"views"`\n\tSegment        string  `json:"segment"`\n\tTags           []Tag   `json:"tags"`\n\tNumberOfImages uint64  `json:"num_images"`\n\tImages         []Image `json:"images"`\n}\n\n// Image corresponds to the data structure\n// of image metadata pulled from PornHub.\n// It references popular comments and tags,\n// as well as general info about popularity.\ntype Image struct {\n\tTimestamp        time.Time `json:"timestamp,omitempty"`\n\tURI              string    `json:"uri,omitempty"`\n\tVotes            uint64    `json:"votes"`\n\tUpvotePercent    float64   `json:"upvote_percent,omitempty"`\n\tViews            uint64    `json:"vites"`\n\tNumberOfComments uint64    `json:"num_comments"`\n\tPopularComments  []Comment `json:"popular_comments,omitempty"`\n\tTags             []Tag     `json:"tags,omitempty"`\n}\n\n// Comment corresponds to the data structure\n// for a PornHub image comment. It contains simple\n// information about the comments. To not reveal\n// too much info about casual users their actual\n// profiles aren\'t crawled\ntype Comment struct {\n\tUsername   string `json:"username,omitempty"`\n\tNetUpvotes uint64 `json:"net_upvotes"`\n\tText       string `json:"text"`\n}\n\n// Tag corresponds to generic information about\n// tags (just the name currently to save space)\ntype Tag string\n```\n\n## Endpoints\n\n#### `GET /`\n##### Status Check\n\nEndpoint returns some generic info about how the crawling is going.\n\nExample Response:\n\n```json\nHTTP/1.1 200 OK\nContent-Length: 104\nContent-Type: application/json\nDate: Sat, 12 Dec 2015 01:07:34 GMT\n\n{\n    "errors": [],\n    "pages_crawled": 2,\n    "saving_json_to": "./data",\n    "status": "running",\n    "uptime": "16m15.553127188s"\n}\n```\n\n# LICENSE (data = PornHub\'s ; code = MIT)\n\nAll data remains within the rights reserved by [PornHub](http://www.pornhub.com/information#terms).\n\nFor the code (not the data):\n\n```\nThe MIT License\n\nCopyright (c) 2015 Conner DiPaolo @cdipaolo\n\nPermission is hereby granted, free of charge, to any person obtaining a copy\nof this software and associated documentation files (the "Software"), to deal\nin the Software without restriction, including without limitation the rights\nto use, copy, modify, merge, publish, distribute, sublicense, and/or sell\ncopies of the Software, and to permit persons to whom the Software is\nfurnished to do so, subject to the following conditions:\n\nThe above copyright notice and this permission notice shall be included in\nall copies or substantial portions of the Software.\n\nTHE SOFTWARE IS PROVIDED "AS IS", WITHOUT WARRANTY OF ANY KIND, EXPRESS OR\nIMPLIED, INCLUDING BUT NOT LIMITED TO THE WARRANTIES OF MERCHANTABILITY,\nFITNESS FOR A PARTICULAR PURPOSE AND NONINFRINGEMENT.
