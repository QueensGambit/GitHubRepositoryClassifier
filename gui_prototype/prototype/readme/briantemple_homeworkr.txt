b'== Homeworkr\n\nHomeworkr is a simple rails app to handle assignment submission and grading.\n\n== Installation\n\n* Install Homeworkr gem dependencies\nbundle install\n\n* Create the database using the command\nrake db:schema:load RAILS_ENV=production\n\n* Define an administrator user\nModify the db/seeds.rb file and change at least the email and password\n\n* Add the administrator user to the database\nrake db:seed RAILS_ENV=production\n\n* Start the server\nrails s --environment=production\n\n* Log in and create courses and assignments\nhttp://localhost:3000\n\n== Contact\n\nProject contact: Brian Temple <brian@briantemple.com>\n\n== License\n\nHomeworkr is released under the MIT license.
