b'# Saudi Newspapers Corpus (SaudiNewsNet)\nThis repo contains a set of **31,030** Arabic newspaper articles alongwith metadata, extracted from various online Saudi newspapers.\n\nFile Structure\n--------------\nThe folder `dataset` contains a set of ZIP files, where each file has the format `YYYY-MM-DD.zip` and contains one JSON file with a corresponding name `YYYY-MM-DD.json`. The JSON files are stored in UTF-8 encoding.\n\nEach JSON file contains an array of articles (the format of each article is explained in the next section), and its file name reflects the date on which the contained articles were extracted.\n\nArticle JSON Object Format\n--------------------------\nThe JSON object for each article contains the following fields (some fileds can have empty values in case the crawler failed to extract them):\n\n - **`source`**: A string identifief of the newspaper from which the article was extracted. It can have one of the following values:\n\n    | String Identifier  | Newspaper |\n    | ------------------ | --------- |\n    | aawsat | [Al-Sharq Al-Awsat](http://aawsat.com/) |\n    | aleqtisadiya | [Al-Eqtisadiya](http://aleqt.com/) |\n    | aljazirah | [Al-Jazirah](http://al-jazirah.com/) |\n    | almadina | [Al-Madina](http://www.al-madina.com/) |\n    | alriyadh | [Al-Riyadh](http://www.alriyadh.com/) |\n    | alwatan | [Al-Watan](http://alwatan.com.sa/) |\n    | alweeam | [Al-Weeam](http://alweeam.com.sa/) |\n    | alyaum | [Al-Yaum](http://alyaum.com/)  |\n    | arreyadi | [Arreyadi](http://www.arreyadi.com.sa/) |\n    | arreyadiyah | [Arreyadi](http://www.arreyadiyah.com/) |\n    | okaz | [Okaz](http://www.okaz.com.sa/) |\n    | sabq | [Sabq](http://sabq.org/) |\n    | was | [Saudi Press Agency](http://www.spa.gov.sa/) |\n    | 3alyoum | [Ain Alyoum](http://3alyoum.com/) |\n\n - **`url`**: The full URL from which the article was extracted.\n - **`date_extracted`**: The timestamp of the date on which the article was extracted. It has the format `YYYY-MM-DD hh:mm:ss`. Notice that this field does not necessarily represent the date on which the article was authored (or made available online), however for articles stamped with a date of extraction after August 1, 2015, this field most probably represents the date of authoring.\n - **`title`**: The title of the article. Can be empty.\n - **`author`**: The author of the article. Can be empty.\n - **`content`**: The content of the article.\n\nStatistics\n----------\nThe dataset currently contains **31,030** Arabic articles (with a total number of **8,758,976 words**). The articles were extracted from the following Saudi newspapers (sorted by number of articles):\n\n - [Al-Riyadh](http://www.alriyadh.com/) (4,852 articles)\n - [Al-Jazirah](http://al-jazirah.com/) (3,690 articles)\n - [Al-Yaum](http://alyaum.com/) (3,065 articles)\n - [Al-Eqtisadiya](http://aleqt.com/) (2,964 articles)\n - [Al-Sharq Al-Awsat](http://aawsat.com/) (2,947 articles)\n - [Okaz](http://www.okaz.com.sa/) (2,846 articles)\n - [Al-Watan](http://alwatan.com.sa/) (2,279 articles)\n - [Al-Madina](http://www.al-madina.com/) (2,252 articles)\n - [Al-Weeam](http://alweeam.com.sa/) (2,090 articles)\n - [Ain Alyoum](http://3alyoum.com/) (2,080 articles)\n - [Sabq](http://sabq.org/) (1,411 articles)\n - [Saudi Press Agency](http://www.spa.gov.sa) (369 articles)\n - [Arreyadi](http://www.arreyadi.com.sa/) (133 articles)\n - [Arreyadiyah](http://www.arreyadiyah.com/) (52 articles)\n\nCiting this Work\n------------------\nIf you\'d like to cite this work, you may use one of the following.
