b'# Destiny.gg\r\nSource code for the website [www.destiny.gg](http://www.destiny.gg/)\r\nThis is a work in progress!\r\n\r\n## License\r\n\r\nSee [LICENSE.md](LICENSE.md)\r\n\r\n## Requirements\r\n\r\n### Building\r\n\r\n[nodejs](http://nodejs.org/) Dependency manager\r\n\r\n[grunt](http://gruntjs.com/) Project builder\r\n\r\n[composer](http://getcomposer.org/) PHP dependency manager\r\n\r\n[glue](http://glue.readthedocs.org/) Glue is a simple command line tool to generate CSS sprites\r\n\r\n### Running\r\n\r\n[nginx](http://httpd.apache.org/), [php 5.5+](http://php.net/), [mysql 5](http://dev.mysql.com/), [Redis](http://redis.io/download)\r\n\r\n\r\n## Getting Started\r\n\r\n\r\nCreate the configuration file "config/config.local.php" and override what you need.\r\n\r\nCreate and load the database using `destiny.gg.sql`\r\n\r\n\r\n### Dependencies\r\n\r\nInstall the node dependencies\r\n\r\n```shell\r\nnpm install\r\n```\r\n\r\nInstall the PHP dependencies\r\n\r\n```shell\r\ncomposer install\r\n```\r\n\r\nBuild the project\r\n\r\n```shell\r\ngrunt build\r\n```\r\n\r\n## The cron job\r\n\r\nAll api requests and heavy tasks are done on a single cron task (currently running every 60 seconds on the live server)\r\nThis is controlled by the Scheduler, by running "Tasks".\r\n\r\nIf you are running the website locally, you can call this file manually, or setup a cron. `/cron/index.php`\r\n\r\nThe table "[prefix_]scheduled_tasks" will show when specific tasks have been run.\r\n\r\nIf you don\'t run this, you will get empty UI and limited functionality in the site.\r\n\r\n\r\n## Grunt Tasks\r\n\r\nBuild the project\r\n\r\n```shell\r\ngrunt\r\n```\r\n\r\nGlue - create the project sprites\r\n\r\n```shell\r\ngrunt glue\r\n```'
