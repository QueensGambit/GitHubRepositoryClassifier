b'## Ember Website\n[![Build Status](https://travis-ci.org/emberjs/website.svg?branch=master)](https://travis-ci.org/emberjs/website)\n\nThe website for the Ember.js project.\n\n### Contributing\n\nTo get started:\n\n``` sh\ngit clone https://github.com/emberjs/website.git\ncd website\nbundle\nbundle exec middleman\n```\n\nThen visit [http://localhost:4567/](http://localhost:4567/)\n\n**Note**: unless you\'re working with the API docs, don\'t use `rake preview`\xe2\x80\x94use `middleman` (aka `middleman server`) instead.\n\nTo populate organizers data from meetup.com on [http://localhost:4567/meetups](http://localhost:4567/meetups)\n\n  * get a [meetup API Key](https://secure.meetup.com/meetup_api/key/)\n  * set `ENV["MEETUP_API_KEY"]`\n  * run the following rake command\n\n``` sh\nrake findorganizers [force=true]\n\nThe force=true flag will overwrite all existing organizer data\n```\n\n#### API Documentation\n\nYou can preview the API documentation by generating the docs from the source code. To begin, ensure\nthat you have the necessary dependencies:\n\n- Node\n- npm\n- Ruby\n- bundler\n\nNext, clone the three repositories: [`ember.js`](https://github.com/emberjs/ember.js),\n[`data`](https://github.com/emberjs/data), and this repository, `website`. The repositories need\nto be placed in the same directory:\n\n    emberjs/\n        website/\n        ember.js/\n        ember-data/\n\nNote that the name of the data project needs to be `ember-data`, not `data`.\n\nNext, navigate into the root directory of this repository and execute `bundle exec rake generate_docs`. This\nwill build the documentation for the `ember.js` and `data` repositories.\n\nYou can launch the website via `bundle exec middleman` to preview the generated docs.\n\n### Requirements\n\nIf the `bundle` command is not found, you can install it with `gem install bundle`.\n\nIf the `bundle` command fails to run, you may need to upgrade your Ruby version. The Ember.js website build requires 1.9.3 or newer (2.0.0 recommended). You can use [RVM](https://rvm.io/) to install it:\n\n``` sh\ncurl -L https://get.rvm.io | bash -s stable\nrvm install 2.0.0\nrvm use 2.0.0\n```\n\n### Troubleshooting tips for Windows devs\n\nFor Windows developers using [RubyInstaller](http://rubyinstaller.org/), you\'ll need to [download the DevKit](http://rubyinstaller.org/downloads) and install it using instructions:\nhttps://github.com/oneclick/rubyinstaller/wiki/Development-Kit\n\nAfter you have a proper install, you can then run:\n``` sh\ngem install bundler wdm tzinfo-data\ngem update listen middleman\n```\n\nOnce Middleman comes up, you\'ll be prompted by Windows Firewall. Click "Allow access" and you\'ll be in business!\n'
