Source code documentation\r\n\r\n------------------------------------------------------------------------\r\n\r\n0. Introduction\r\n\r\nBakery is a framework to compute solutions for the shape packing problem\r\nas described by Bennell and Oliveira (J. A. Bennell and J. F. Oliveira. \r\nA tutorial in irregular shape packing problems. The Journal of the \r\nOperational Research Society, 60:s93\xe2\x80\x93s105, 2009).\r\n\r\nBakery was the winning entry of the InformatiCup 2016. For more \r\ninformation see: \r\nhttp://informaticup.gi.de/startseite/rueckblickearchiv/informaticup-2016.html\r\n\r\n------------------------------------------------------------------------\r\n\r\n1. Dependencies\r\n\r\nTo build and run Bakery Qt 5.4 or later is required. See \r\nhttps://www.qt.io/ for more information.\r\n\r\n------------------------------------------------------------------------\r\n\r\n2. Building\r\n\r\nThe recommended way to build Bakery is to use QtCreator. Open\r\n"bakery.pro". If not already done configure the project by selecting\r\na kit and press the button "Configure Project". Use the "Build All"\r\ncommand to start the building process.\r\n\r\n------------------------------------------------------------------------\r\n\r\n3. Graphical user interface\r\n\r\nYou can start the GUI from QtCreator by chosing run target "gui" (click\r\non the computer in the left panel) and executing the "Run" command.\r\n\r\n------------------------------------------------------------------------\r\n\r\n4. Command-line interface\r\n\r\nThere is no convenient way to use the command-line interface from\r\nQtCreator. To be able to start the binary from the command\r\nline libbakery may need to to be added to the linker\'s library path\r\non Mac, Linux:\r\n\r\nMac:\r\n    export DYLD_LIBRARY_PATH="$DYLD_LIBRARY_PATH:[path to libbakery]"\r\n    \r\nLinux:\r\n    export LD_LIBRARY_PATH="$LB_LIBRARY_PATH:[path to libbakery]"\r\n    \r\nOn Windows Qt has to be in the PATH:\r\nWindows:\r\n    set PATH=%PATH%;C:\\Qt\\Qt5.5.1\\5.5\\mingw492_32\\bin\r\n    (if the installed version is 5.5.1 (MinGW32))\r\n    \r\nBy default there is no time limit. To impose a time limit use the\r\n-t/--time-limit parameter. A full list of available parameters may\r\nbe obtained by using the switch -h/--help.\r\n\r\n------------------------------------------------------------------------\r\n\r\n5. Source code documentation\r\n\r\nThe source code documentation can be build by using Doxygen\r\n(doxygen.org). By default doxygen creates output as html, latex and \r\nQt helpfile.\r\n\r\nTo generate the documentation switch the working directory to the bakery\r\ndirectory and use the following command:\r\n   doxygen\r\n'
